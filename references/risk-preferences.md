# 风险偏好与策略

## 保守型

### 选股倾向
- A+H 折价大（>30%）的标的
- 基石投资者阵容强（知名机构、数量多）
- 头部保荐人（高盛、中金等）
- 入场费适中（3000-5000）
- 有绿鞋机制

### 回避
- 入场费过高（>7000）
- 行业冷门或政策不明朗
- 基石投资者少或无名
- 无绿鞋机制
- 集资用途大比例还债

### 策略
- 宁可错过，不要踩雷
- 只打确定性高的标的
- 不融资，只用现金
- 每月可能只打 1-2 只

---

## 均衡型

### 选股倾向
- 综合评分较高的标的
- 不要求每项都完美，但无明显硬伤
- 热门标的可以参与，但不 all in

### 策略
- 分散配置，同期多只新股可以都打
- 热门股现金 + 适度融资
- 冷门股只现金 1 手试试
- 关注性价比：中签率 × 预期收益

---

## 激进型

### 选股倾向
- 追热门赛道和高关注度标的
- 接受高风险高回报
- 入场费高也敢打

### 策略
- 热门股积极融资，提高中签率
- 可以接受偶尔亏手续费
- 看好就多账户打
- 暗盘积极操作

---

## 融资打新分析

### 券商手续费参考

| 券商 | 现金认购手续费 | 融资认购手续费 |
|---|---|---|
| 长桥 | 0 | ~99 港元 |
| 富途 | 0 | ~100 港元 |

> 具体费用以券商最新公告为准，可能随活动调整。

### 融资的本质

- 融资 = 借钱打新，提高认购手数 → 提高中签率
- 代价：不管中不中签，都要付融资手续费（利息）
- 核心问题：**提高的中签概率带来的期望收益，能否覆盖手续费？**

### 期望收益计算

```
期望收益 = 中签率 × 每手预期利润 - 手续费

现金 1 手：
  期望收益 = 中签率_1手 × 预期利润 - 0

融资 N 手：
  期望收益 = 中签率_N手 × 预期利润 - 手续费
```

### 例子

假设某股：
- 每手预期利润：2000 港元
- 现金 1 手中签率：30%
- 融资 5 手中签率：60%
- 融资手续费：99 港元

```
现金 1 手期望收益 = 30% × 2000 - 0 = 600 港元
融资 5 手期望收益 = 60% × 2000 - 99 = 1101 港元
```

融资多赚 501 港元，值得。

### 什么时候融资划算

融资划算的条件：
1. **预期利润高**：首日涨幅预期 > 10%
2. **中签率提升明显**：融资后中签率显著高于现金
3. **手续费相对低**：99 元相对预期利润占比小
4. **标的确定性高**：基本面好、市场热度高

融资不划算的情况：
1. **冷门股**：中签率本来就高（现金也能中），融资没意义
2. **预期利润低**：涨幅预期 < 5%，中了也赚不多
3. **破发风险大**：中签反而亏钱，融资放大亏损

### 简单判断规则

```
如果 预期利润 × (融资中签率 - 现金中签率) > 手续费 × 2
  → 融资划算
否则
  → 不融资
```

> 乘以 2 是留安全边际，因为预期利润本身就是估算值。
